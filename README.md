# Customer Churn Prediction using Machine Learning

## Project Overview

This project uses machine learning to predict customer churn based on customer behavior data. The goal is to identify customers who are likely to leave and support retention-focused business decisions.

## Dataset

The dataset includes customer-related features such as:

* Age
* Gender
* Tenure
* Usage frequency
* Support calls
* Payment delay
* Subscription type
* Contract length
* Total spend
* Last interaction

## Model

* Logistic Regression
* Accuracy: ~89.6%

## Prediction Results

The model identified approximately 47,981 customers as likely to churn.

Among them, 40,312 customers were classified as very high risk, with a churn probability above 80%.

## Key Insights

* The model achieved strong predictive performance.
* Customer behavior such as tenure, usage frequency, and total spend plays a major role in churn prediction.
* Probability-based predictions help prioritize high-risk customers.
* This approach can support customer retention strategies and better business decision-making.

## Tools Used

* Python
* Pandas
* Scikit-learn
* Google Colab

## Business Value

This project demonstrates how machine learning can be used to identify at-risk customers and support proactive retention strategies.
